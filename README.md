# Chat en tiempo real con .NET Core y SignalR
Sitio web con patrón de diseño MVC para un chat en tiempo real, realizado con .NET Core, SignalR, bootstrap y Docker.

# Caracteristicas
- Docker
- Chat en tiempo real
- Patrón de diseño MVC
- Múltiples salas
- Bootstrap
